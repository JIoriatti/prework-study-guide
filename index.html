HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Elements are enclosed in tags, with the ending tag having a backslash in front of it.
  • Child elements are sub elements within a parent element.
  • The head and body elements are semantic elements.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Colors can be added in CSS using semantic references or by using hexadecimal code.
  • Placing a period before a class will denote a class selector, allowing property assignments to be applied to all HTML elements assigned to that class.
  • External CSS files must be linked to the HTML file for it's properties to be applied to the file.
  • rel and href are attributes that specify relationship between current document and linked document resource, and location of the external resource, resepctively.

Git

  • Git commands start with 'git'.
  • 'git checkout -b' followed by branch name will create a new branch of the name given.
  • 'git status' will tell you which branch you are currently on.

JavaScript

  • A function must be called in order to work.
  • An array can be used to store many values in one variable.
  • For loops are used to run through a specific segment of code multiple times.
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
